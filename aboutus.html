About steducation


What is steducation

STeducation is a web developers site, with tutorials and references on web development languages such as HTML, CSS and JavaScript, etc.
The site derives its name from the short-name of the website developers, but is not affiliated with the W3.
STeducation was originally created in 2005 by K.S.T, a Norwegian software development and consulting company.


Easy Learning

Steducation has focus on simplicity.
Steducation practice easy and straight-forward learning.
Steducation uses simple code explanations with simple illustrations of how to use it.
Steducation's tutorials start from basic level and move all the way up to professional references.


Steducation is Free

Steducation is free education website, and will always be, a completely free developers resource.
